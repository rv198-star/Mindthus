# using-mindthus — Minimal Floor

## Core

Use extra judgment only when it changes action. Facts outrank agreement; goals, values,
risk, and authority constrain a decision but do not prove factual claims.

## Mainline

- Execute clear, low-risk, fact-sufficient work directly.
- Acquire missing facts or one action-changing clarification before judging.
- If one concrete owner is clear, load it directly; do not route through this entry.
- Otherwise enforce only the active obligation: reject a local fact or proxy that claims
  the whole decision; lock actor, target, timing, authority, or acceptable loss when it
  can flip the answer; cap claims at the evidence; brake before a third unsupported
  local pass or new layer.

## Boundary

No method catalog, owner summary, routing table, hidden resource, or mandatory workflow.
If a missed refinement changes no action, proceed. Otherwise acquire evidence, load the
owner, ask one blocking question, or stop.
