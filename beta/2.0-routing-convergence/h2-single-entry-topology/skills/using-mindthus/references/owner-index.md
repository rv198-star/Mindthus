# Owner Index

Read only after the entry's Decision Context and Evidence First gates are satisfied and
a hard judgment remains. Read exactly one matching `OWNER.md`; no match means direct.

| Owner | Independent gate | Hard exclusion | Resource |
| --- | --- | --- | --- |

| EDSP | A qualitative proposition is structurally ambiguous: A/B both look right, the binary may be malformed, or an extreme coordinate changes the judgment. | Missing facts require evidence first; an executable known problem does not need EDSP. | `owners/edsp/OWNER.md` |

| WAE | An LLM/agent/skill/prompt/workflow system has a real controller mismatch among workflow, agentic judgment, evidence, script, schema, or authority. | Ordinary product process, release readiness, or generic evidence sufficiency is outside WAE. | `owners/wae/OWNER.md` |
| TVG | A bounded AI artifact already exists and has a named value gap requiring targeted strengthening. | Do not reopen strategy, invent an artifact, or use TVG as generic review. | `owners/tvg/OWNER.md` |
| TPlan | An explicitly authorized durable Mission needs script-managed task state, acceptance evidence, authority, recovery, or closure. | Ordinary planning and task lists do not need a Mission runtime. | `owners/tplan/OWNER.md` |
