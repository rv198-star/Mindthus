可以按“只动价格权重、其他参数冻结”的方式做，避免把前 3 轮收益冲掉。

**步骤**

1. **固定基线**
   - 以第三轮配置作为基线：满意度 `72%`
   - 低库存、新品、噪声标签权重先不变

2. **确认价格信号方向**
   - 不要简单理解为“价格越低越好”
   - 建议价格特征拆成：
     - 用户偏好价格带匹配度
     - 同类商品价格竞争力
     - 折扣/性价比
     - 过低价或异常价惩罚

3. **做离线回放**
   - 设当前价格权重为 `W`
   - 跑几组候选：
     - `W * 1.05`
     - `W * 1.10`
     - `W * 1.15`
     - `W * 1.20`
   - 观察满意度预估、点击率、转化率、客单价、低质低价商品曝光占比

4. **选 2-3 个线上实验组**
   - A 组：当前第三轮配置，作为对照
   - B 组：价格权重 `+10%`
   - C 组：价格权重 `+15%`
   - 如离线表现稳定，可加 D 组：`+20%`

5. **小流量灰度**
   - 先放 `5%-10%` 流量
   - 至少跑满一个完整业务周期，比如 24-48 小时
   - 主指标：满意度是否从 `72%` 接近或超过 `75%`
   - 保护指标：转化率、GMV、退款率、低库存曝光、新品曝光、内容多样性

6. **根据结果微调**
   - 如果满意度上升且保护指标稳定：继续提高 `5%`
   - 如果点击上升但满意度不升：说明低价吸引点击但体验不足，需要加质量门槛
   - 如果 GMV 或客单价明显下滑：降低价格权重，改为加强“价格带匹配”而不是“低价优先”
   - 如果低价商品过度曝光：增加异常低价惩罚或类目内归一化

7. **达标后放量**
   - 满意度稳定到 `75%` 左右，且保护指标无明显回退，再从 `10% -> 30% -> 50% -> 100%`
   - 每一步观察至少 24 小时

8. **设置回滚线**
   - 满意度低于 `72%`
   - 或转化率/GMV/退款率任一关键指标明显恶化
   - 立即回退到第三轮配置

建议今天先从 `+10%` 和 `+15%` 两个价格权重实验开始，目标是用最小改动拿到从 `72%` 到 `75%` 的增量。
