# Anti-Spiral / 反螺旋

## Role

Anti-Spiral prevents local repair from swallowing the Mission. It activates when the
agent keeps working the same small object while the real target stops moving.

Full method resource: [Anti-Spiral Self-Audit](../anti-spiral-self-audit.md).

## Short Rule

同一局部对象第三次、负反馈或加层冲动出现时，先停下回看上游。

## When It Intervenes

- Before same-path continuation.
- When the same local object is touched for the third time.
- When user feedback gets worse.
- When the next idea is to add another layer instead of moving the core target.

## Action Effects

- Stop the local loop.
- Return upstream to Mission, target, or acceptance evidence.
- Subtract or replace instead of adding structure.
- Ask for user authority before continuing.

## Must Not

- It must not block a first legitimate iteration.
- It must not become a new project-management method.
- It must not hide ordinary uncertainty behind a dramatic stop.

## Probe

Am I improving the Mission, or just making the same local object look more managed?
